# Libasora Clothing Website

`Libasora` aik unique clothing brand name hai jo premium aur modern fashion feel deta hai.

## Files

- `index.html` complete website structure
- `styles.css` full responsive styling
- `script.js` add to cart, checkout, and interactive behavior

## How To Open

1. `C:\Users\Malik\Desktop\fation\index.html` ko browser me open karein.
2. Products par `Add to Cart` click karein.
3. Cart se checkout par jayein.
4. Payment method select karke order place karein.

## Included Features

- hero section
- featured products
- add to cart
- quantity update and remove
- checkout and payment section
- order summary
- newsletter form
- testimonials
- responsive mobile layout
